# OpenHire

Open-source AI-powered interview platform. Self-host on your infrastructure.

## Why OpenHire?

- **100% Self-Hostable** - Run on your own infrastructure
- **Privacy-First** - Candidate data never leaves your servers
- **No Vendor Lock-in** - Swap voice providers anytime
- **Open Source** - MIT licensed, community-driven

## Tech Stack

| Component | Technology |
|-----------|------------|
| Framework | [Next.js 16](https://nextjs.org) (App Router) |
| UI | [shadcn/ui](https://ui.shadcn.com) |
| Auth | [Better Auth](https://better-auth.com) |
| Database | [PostgreSQL](https://postgresql.org) + [Drizzle](https://orm.drizzle.team) |
| AI | [Vercel AI SDK](https://sdk.vercel.ai) (multi-provider) |
| Voice | [VAPI](https://vapi.ai) |

## Features

- AI-powered voice interviews
- Auto-generated interview questions
- Candidate scoring & analytics
- Full transcript & recording
- Self-hostable (own your data)
